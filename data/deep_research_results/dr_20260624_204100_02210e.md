## 4. 手机芯片业务

### 4.1 市场地位

- **Android 高端**：Snapdragon 8 Elite / 8 Gen 3 仍主导三星 Galaxy S 系列、小米/OPPO/vivo 旗舰（除部分联发科份额）。
- **内容提升**：5G Advanced、Wi-Fi 7、卫星通信、AI NPU 集成 → 单机 ASP 长期向上。
- **短期逆风**：内存涨价压制中低端与中国市场建库；Q3 手机芯片指引仅 ~$4.9B。

### 4.2 与 Apple 关系

- 2026 秋 iPhone：Qualcomm 基带份额降至 **20%**，此后 **零产品供应**。
- 专利授权（QTL）规模短期不变，待重新谈判——授权与芯片分开。
- Apple 自研 C1/Ganymede modem 路线明确，2028 目标全面内部化。

### 4.3 竞争格局

| 对手 | 威胁 |
|------|------|
| 联发科 | 中低端侵蚀 + 部分旗舰渗透 |
| 苹果自研 | 高端份额永久流失 |
| 三星 Exynos | 区域有限 |
| 紫光展锐等 | 中国政策/国产化 |

**结论**：手机基本盘仍强但进入「存量博弈 + 内容提升」阶段，不再是增长引擎。

## 6. ARM 诉讼进展

### 6.1 Arm 诉 Qualcomm/Nuvia（C.A. 22-1146）— **Qualcomm 全面胜诉**

| 时间 | 事件 |
|------|------|
| 2024-12 | 陪审团裁定 Qualcomm 未违反 Nuvia ALA，CPU 核合法授权 |
| 2025-09-30 | 法院终局判决：Nuvia 未违约、Qualcomm 已获 ALA 授权 |
| 2025-10+ | Arm 表示将 **上诉** |

**意义**：Oryon CPU（Nuvia 遗产）法律风险大幅解除，PC/服务器/数据中心 CPU 路线获得许可证保障。

### 6.2 Qualcomm 诉 Arm（C.A. 24-490）— **进行中**

- 指控：Arm 违约、干扰客户关系、反竞争行为。
- 庭审：原排期 2026-03，法院文件显示可能延后至 **2026-10** 陪审团审判。
- **风险**：若 Arm 在上诉或反诉中限制架构授权条款，可能影响长期 Oryon 路线图——概率低但非零。

### 6.3 投资含义

- 短期：Nuvia 案胜诉 = **利好**，消除最大法律 overhang。
- 中期：Arm 与 Qualcomm 从「合作伙伴」转向「竞争对手+诉讼对手」，需监控授权续约（2030 年代）。

## 7. AI 端侧推理前景

### 7.1 技术栈

| 组件 | 能力 |
|------|------|
| Hexagon NPU（6th Gen） | Snapdragon X2：**80–85 TOPS**（vs X Elite 45 TOPS） |
| 异构计算 | NPU + Adreno GPU + Oryon CPU + Sensing Hub |
| 软件 | QNN/QAIRT 运行时、PyTorch/ONNX/TensorFlow 支持；100+ AI 优化 App |
| 累计出货 | **200 亿+ AI 核心** |

### 7.2 应用场景

- **手机**：Galaxy S26 Ultra「for Galaxy」联合优化、实时翻译、Agentic AI 本地执行。
- **PC**：Copilot+ PC 阵营核心供应商；NPU 性能领先 Intel Lunar Lake / AMD 部分 SKU。
- **汽车/IoT**：舱内感知、ADAS 推理、工业边缘视觉。
- **战略叙事**：CEO Amon 提出 **「AI 是新 UI」**——端侧 Agent 驱动平台换代。

### 7.3 竞争定位

| 维度 | Qualcomm | 对手 |
|------|----------|------|
| 端侧 NPU 性能/瓦特 | **领先** | Apple M 系列（MLX 生态）、Intel/AMD 追赶 |
| LLM 端侧推理 | 可用但受内存带宽限制 | Apple 128GB 统一内存方案更优 |
| 开发者生态 | QNN 快速成熟 | Apple Core ML、Intel OpenVINO |

**结论**：端侧 AI 是高通 **最确定的差异化优势**，与数据中心故事形成「边缘→云」闭环。
