# 13-Master Lite · 价格纪律

> 适用于飞书/Cloud Agent 单票深研；完整版见 `skills/13master-stock-research/references/price-verification.md`。

## 三源验价（Lite 版）

| 源 | 要求 | Lite 深研 |
|----|------|-----------|
| A | Web 财经站 1（Yahoo / Google Finance 等） | 必须，写入报告 |
| B | Web 财经站 2（Investing / Morningstar 等，须与 A 不同） | 必须，写入报告 |
| C | 盈立截图 / 用户报现价 | **深研可缺**；缺 C 时禁止输出 GTC 挂单清单，操作价位须标注「待第三源验价」 |

## 52 周高点纪律

每票必须展示：

- 现价（双源 + 时间戳 + USD）
- 52 周高点
- 距 ATH 百分比
- 近 30 日 ±10% 单日异动（若有）

## 进场措辞

| 条件 | 输出 |
|------|------|
| 距 ATH ≤5% 或近 1 月涨幅 >50% | 禁止写「进场区间」；改写「等回调到 $X–Y」 |
| 距 ATH 6–15% | 可写保守进场区，标注 ATH 警示 |
| 距 ATH >15% 且基本面通过 | 可写「立即可建」进场区 |

## 仓位上限（相对核心仓）

无 `holdings.yaml` 核心仓时，默认以「单票最大风险敞口」表述：

| 共识档位 | 仓位上限 |
|----------|----------|
| 立即可建 | 核心仓 × 1/3（或组合 5% 名义上限） |
| 等回调 | 核心仓 × 1/5 |
| 跳过 | 0 |

有 `data/portfolio_health.md` 健康度警示时，整体下调一档。

## 止损 / 卖出触发（框架价，非挂单）

- **硬止损**：进场下沿 −8%～−12%（高波小盘取宽）
- **软警报**：−5% 暂停加仓
- **卖出触发**：① 硬止损 ② 叙事破坏（财报 miss / 管理层变动）③ 达 TP 档位（按 categories 默认策略，仅写框架价）

所有美元操作价须脚注：**⚠️ 待盈立第三源验价后方可执行**

## 报告必含字段

1. 验价表（A/B，可选 C 状态）
2. 52w 高点与 ATH 距离
3. 13-Master 六派系分 + 校正后均分
4. 三档结论：立即可建 / 等回调 / 跳过
5. 进场区 / 仓位上限 / 止损 / 卖出触发
6. 与现有持仓重叠度（读 `config/holdings.yaml`）
